⚙️ Cài đặt

📋 Hướng dẫn lấy API key:
1. Truy cập console.anthropic.com
2. Đăng ký / đăng nhập tài khoản
3. Vào mục API Keys → Create Key
4. Copy key (bắt đầu bằng sk-ant-) và dán vào ô trên
5. Nhấn Lưu — key được lưu trong máy, không gửi đi đâu khác
⚠️ Lưu ý: sử dụng API sẽ tốn phí theo lượt dùng (~$0.003/lần tra cứu sao). Xem chi phí tại anthropic.com/pricing
